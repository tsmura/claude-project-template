# Create Component

Create a new component named $ARGUMENTS:

1. Create the component source file in the appropriate `src/` subdirectory
2. Create a corresponding test file in `tests/`
3. Export the component from the relevant index file if one exists
4. Follow existing patterns in the codebase for file structure and naming
